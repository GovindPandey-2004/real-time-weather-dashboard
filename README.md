# 🌦 WeatherScope – Real-Time Weather Dashboard

A modern weather dashboard built using HTML, CSS, and JavaScript that fetches live weather data using the OpenWeather API.

## Features

- 🌍 Search weather by city
- 🌡 Live temperature
- 💧 Humidity
- 💨 Wind speed
- ☁ Dynamic weather icons
- ⚡ Real-time API integration
- 📱 Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API

## Installation

1. Clone this repository

```
git clone https://github.com/GovindPandey-2004/real-time-weather-dashboard.git
```

2. Open index.html in your browser.
